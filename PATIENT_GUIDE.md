## Account Management

### Patient Profile
- **View Profile**: Patients can view their own profile information including:
  - Personal details (name, age, gender)
  - Contact information
  - Medical history summary
  - Allergies and medications
  - Emergency contact information

- **Update Profile**: Patients can update their personal information, contact details, and basic medical information.

### Authentication
- **Sign Up**: Create a new patient account with email and password
- **Login**: Access the platform with registered credentials
- **Password Reset**: Request password reset via email if forgotten
- **Logout**: Securely log out from the platform

## Doctor Appointments

### Booking Appointments

#### Search Methods
Patients can book appointments using two search methods:
1. **Search by Doctor**: Browse and select from available doctors
2. **Search by Medical Center**: Browse centers and select doctors from those centers

#### Appointment Types
- **Clinic Consultation**: Book appointments at medical centers/clinics
- **Home Visit**: Request doctor visits at patient's home address (if doctor offers this service)

#### Booking Process
1. Select appointment type (clinic or home visit)
2. Choose search method (by doctor or by center)
3. Search and filter doctors/centers by:
   - Specialty
   - Location
   - Availability
   - Consultation fees
4. Select a doctor
5. Choose a medical center (for clinic appointments) or provide home address (for home visits)
6. Select available date and time slot
7. Add notes or special requirements
8. Confirm booking

#### Appointment Information
Each appointment includes:
- Doctor name and specialty
- Date and time
- Duration
- Location (center address or home address)
- Appointment type (clinic/home visit)
- Status (scheduled, confirmed, completed, cancelled)
- Contact information
- Notes

### Managing Appointments

#### View Appointments
- **My Appointments Page**: View all appointments in one place
- **Filtering Options**:
  - By status (all, scheduled, confirmed, completed, cancelled)
  - By type (all, clinic, home visit)
  - By date range (from date, to date)
  - By search text (doctor name, specialty)
- **Sorting**: 
  - Upcoming appointments shown first (earliest date)
  - Completed/cancelled appointments shown with latest first

#### Appointment Actions

**For Scheduled/Confirmed Appointments:**
- **Reschedule**: Change appointment date and/or time
  - Select new date and time from available slots
  - Provide reason for rescheduling (optional)
- **Cancel**: Cancel the appointment
  - Provide cancellation reason (optional)
  - Cannot cancel completed appointments

**For Completed Appointments:**
- **Leave Review**: Rate and review the doctor
  - 1-5 star rating system
  - Optional written comment
  - Can only review once per appointment
- **View Visit Summary**: Access detailed visit information
  - Diagnosis
  - Treatment provided
  - Prescriptions
  - Lab results (if any)
  - Medical notes

#### Appointment Status
- **Scheduled**: Appointment is booked but not yet confirmed
- **Confirmed**: Appointment is confirmed by the system/doctor
- **Completed**: Appointment has been completed
- **Cancelled**: Appointment was cancelled
- **Absent**: Patient missed a scheduled appointment (marked automatically)

## Lab Tests & Medical Imaging

### Finding Labs & Centers

#### Browse Medical Centers
- **Patient Dashboard - Labs Section**: Access dedicated labs page
- **Search & Filter**:
  - Search by center name or location
  - Filter by service type:
    - Lab Tests
    - Imaging Services
    - All Services
- **View Center Details**:
  - Center name and address
  - Contact information (phone, email)
  - Operating hours
  - Available services
  - List of available tests with descriptions and fees

### Booking Lab Tests

#### Booking Process
1. Browse or search for medical centers offering lab/imaging services
2. Select a center
3. View available tests:
   - Lab tests (blood tests, urine tests, etc.)
   - Imaging services (X-rays, MRIs, CT scans, etc.)
4. Select test type(s)
5. Choose date and time from available slots
6. Add notes or special requirements
7. Review fees
8. Confirm booking

#### Test Information
Each test booking includes:
- Test name and category (lab/imaging)
- Center name and address
- Booking date and time
- Duration
- Fee
- Status
- Preparation notes (if required)
- Result file URL (after completion)

### Managing Lab Bookings

#### View Lab Bookings
- **My Labs Page**: View all lab test bookings
- **Filtering Options**:
  - By status (all, scheduled, confirmed, completed, cancelled)
  - By category (all, lab tests, imaging)
  - By date range
  - By search text (test name, center name)

#### Lab Booking Actions

**For Scheduled/Confirmed Bookings:**
- **View Preparation Instructions**: Access preparation notes if required
- **Reschedule**: Change booking date and/or time
  - Select new date and time
  - Provide reason (optional)
- **Cancel**: Cancel the booking
  - Provide cancellation reason (optional)

**For Completed Bookings:**
- **View Results**: Access test results
  - View result file (PDF or image)
  - Read result notes
  - View result date
- **Download Results**: Download result files as PDF

## Access Restrictions

### What Patients CANNOT Do

1. **Cannot Access Other Users' Data**:
   - Cannot view other patients' appointments or medical records
   - Cannot access doctor dashboard features
   - Cannot access center/admin management features

2. **Cannot Modify Certain Data**:
   - Cannot delete medical records created by doctors
   - Cannot modify appointment details after completion
   - Cannot change lab test results

3. **Cannot Perform Administrative Actions**:
   - Cannot manage doctor schedules
   - Cannot add or remove medical centers
   - Cannot manage system settings
   - Cannot access analytics or reports (except their own)

4. **Limited Appointment Actions**:
   - Cannot reschedule or cancel completed appointments
   - Cannot reschedule or cancel already cancelled appointments
   - Cannot book appointments for other patients

5. **Review Limitations**:
   - Can only review each appointment once
   - Cannot edit or delete reviews after submission
   - Cannot review appointments that are not completed

### Security & Privacy
- Patients can only access their own data
- All actions require authentication
- Profile updates are restricted to the patient's own profile
- Medical records can only be viewed/updated by the patient or their treating doctors

## Getting Started

### For New Patients:
1. **Sign Up**: Create an account with email and password
2. **Complete Profile**: Add personal information and medical history
3. **Explore**: Browse doctors and medical centers
4. **Book Appointment**: Schedule your first appointment
5. **Manage**: Use the dashboard to manage appointments and records

### For Existing Patients:
1. **Login**: Access your account
2. **Dashboard**: View upcoming appointments and recent records
3. **Book Services**: Schedule new appointments or lab tests
4. **Manage**: Update profile, reschedule, or cancel as needed
5. **Review**: Leave feedback after completed visits
